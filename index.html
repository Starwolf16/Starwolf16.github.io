Paragon

Paragon is a MOBA (Massive Online Battle Arena) game that was created by the same developers that made
Infinity Blade.

Here is a list of the things that are cool about Paragon:
  • The graphics of the game are really good compared to some of the other games that are out right now
  • The game is currently still in development, and this means that if you don't like it now, wait for a new update and see if you like it then
  • This game has potential to do well, and many people will try it because of the genre of game that it is.
  • The developers of Paragon have promised that there will be a new hero to play every 3 weeks, allowing players to experience something different
